# Aleks's Dotfiles

Personal configuration files for macOS.

## Configuration

- **Shell**: Zsh with [Oh My Zsh](http://ohmyz.sh/)
- **Editor**: [Visual Studio Code](https://code.visualstudio.com/)
- **Version Control**: [Git](https://git-scm.com/)

## Installation

You can install these dotfiles with a single command:

```bash
sh -c "$(curl -fsSL https://raw.githubusercontent.com/a160v/dotfiles/master/setup.sh)"
```

Or manually:

```bash
git clone git@github.com:aleks/dotfiles.git ~/dotfiles
cd ~/dotfiles
./install.sh
```
